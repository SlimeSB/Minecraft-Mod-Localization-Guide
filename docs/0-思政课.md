# 思政课

我们的工作场所正从相对封闭的内部空间，转变更广阔的公共空间；从相对隐性的状态，转向公开透明的舞台。站在阳光之下，我们面对的是来自多方的注视——既有普通受众的目光，也有监管机构的审视。在这样的环境中工作，意味着我们必须学会在开放与监督中前行，这也对我们提出了新的要求：具备明辨是非的清醒，或者说，一种基础的政治素养。对译者而言，这不仅是避免争议与批评的盾牌，更是职业道路上稳健前行的保障。

这里所说的“政治素养”，并非单纯指课堂教材中的理论知识。作为最早接触并传递外来文化的人群之一，译者往往处于中外思想交汇的前沿。面对外来观念与表达，我们既要保持开放，也需持有审慎的辨识力。然而，政治本身是一个复杂且时常敏感的领域，在相关实践中，话题的边界与表达的分寸都需要谨慎把握。因此，在接下来的讨论中，若我的笔触偶显克制或迂回，还望读者理解。一方面是不适宜展开，一方面是笔者能力确实有限。

## 底线

在翻译实践中，译者始终面临一个根本性的定位问题：我们究竟应当为谁传达声音？是忠实于原文作者，还原文本本意，还是服务于译入语读者？事实上，译者的角色并非单一传声筒，而往往是在作者意图、文本内容、读者接受与社会语境之间寻求审慎平衡的协调者。

然而，当遇到具有争议性、涉及是非判断，尤其是触碰原则底线的极端内容时，译者必须保持清醒的立场。在这种情况下，我们应当自觉站在维护国家利益、社会公共利益和人民群众根本利益的立场上。这不仅是一种责任，也是译者自我保护、避免陷入法律与道德风险的必然选择。

那么，究竟什么样的内容属于需要警惕的“问题”呢？我认为，应当区分作品的**表现形式**与**精神内核**。例如，恐怖题材作为一种艺术表现形式，其本身并不必然构成问题。国内游戏市场也需要类型多样的作品，包括恐怖类游戏，这是文化创作多样性的正常体现，不必过度敏感或“上纲上线”。

真正需要译者警惕的，是那些表面形式正常、但内核传递有害或具有煽动性价值观的内容。比如某些作品可能借普通叙事包装，宣扬极端思想、扭曲历史、破坏社会共识，或隐含政治误导。这类内容往往更具隐蔽性与危害性。译者在接触此类内容时，应具备识别与判断的能力，并在必要时通过翻译策略做出审慎处理，甚至进行有意识的筛选与引导，避免其通过翻译渠道产生不良影响。

因此，译者的政治素养不仅体现在“知道该翻译什么”，更体现在“懂得如何翻译”以及“明白为何这样翻译”。它要求我们在复杂的内容与语境中，既保持专业与开放，也坚守底线与立场。

## 世界主义

在探讨翻译的跨文化实践时，我们可以引入一种世界主义的视野。这里所说的世界主义，并非某种具有严格界定的理论体系，而更接近于一种翻译伦理与精神取向：它倡导在文化交往中真诚地**尊重彼此的习俗与传统**，帮助读者超越自身文化的局限，以更开放的眼光看待差异；它呼唤一种具有普遍关怀的伦理视角，关注**人类共同的命运与价值**；它自觉**反思并抵制任何形式的文化霸权**，无论其来自何方；最终，它指向一种建设性的理想——通过语言的转渡与思想的沟通，促进不同文明之间的理解与共生，**共同迈向一个更具包容性、更美好的未来**。

### 引进来

#### 文化专有项

文化专有项指源语文化中特有、译入语文化中缺失的概念、物品、习俗等。学界对其处理方式大致可分为两种策略：

- **归化**：使译文贴近译入语文化的表达习惯，降低读者的理解障碍。
- **异化（新造）**：保留原文的文化特色与形式，让读者感受到“异质性”，促进文化输入。

当前围绕这两种策略存在一些讨论：有人认为“归化”是对他者文化的不尊重，不符合世界主义精神；而“异化”则常被批评为生硬难懂，违背语言习惯，乃至殖民主义复辟。

笔者认为，翻译是译者主观能动性较强的跨文化实践。在处理文化专有项时，没有绝对“正确”的方法，更多取决于译者的判断与目标语境。归化不是简单的“文化不自信”，而可以是一种促进理解的妥协、变通；异化也不应被等同于“生硬晦涩”，它是语言与文化发展的潜在动力。许多今天看来自然的译词，如“幽默”“俱乐部”，都曾经历从陌生到接纳的过程。翻译的生命力常需时间检验，最终由语言自身的演化与读者的接受来完成筛选。

#### 东方主义

东方主义指西方长期以来对东方文化形成的刻板化、想象性建构，例如某些作品中将东方元素简化为奇观符号，或沿用陈旧的文化定型（如清朝僵尸、旗袍女子等固定形象）。这类内容有时隐含文化偏见，甚至构成歧视。

译者在处理此类文本时应当保持清醒：

- **对明显歧视性、侮辱性内容**，译者请务必积极表达诉求，要求对相关内容进行中立化修改。否则可以拒绝提供服务。
- **对无恶意、已符号化且无伤大雅的文化元素**，则不必过度敏感或强行“净化”，可在注释或译序中加以说明，引导读者理性看待。

#### 汉字文化圈

汉字文化圈（主要是日本、韩国、越南）的文本处理具有其特殊性与复杂性。这些国家的语言在历史上深受汉语影响，至今仍保留着大量汉字或汉字衍生词汇，形成了独特的“汉源词”。与此同时，各国语言中也存在“固有词”（本土原生词汇）与“假借词”（主要来自西方语言的音译词）。这一语言背景使得相关翻译在“可直译”与“需意译”之间往往存在微妙界限。

##### 汉源词

多数汉源词在含义上与汉语一致或高度对应，可直接转写为汉字使用，如“哲学”“革命”等和制汉语，已成为中文的一部分。地名翻译尤其典型：日、韩、越绝大多数地名都有对应汉字形式，且为官方所承认，翻译时宜采用这些既定汉字，而非音译。

然而，部分汉源词虽字形相同，含义却已分化，甚至截然不同。例如日语“大丈夫”意为“没关系”，“情報”虽可理解，但中文更常用“信息”。这类词汇需按其在本语中的实际含义翻译，不可机械照搬汉字。

##### 固有词与假借词

固有词有一部分是文化专有项，是中文没有对应物的。对于一般的固有词根据词典翻译就好。对于词典没有的难译项，可以参考翻译指南中对[难译项](https://cfpa.site/TransRules/#5-难译项)的翻译策略。

假借词一般按其在该国语言中的实际含义处理，源语言形式可作为参考，但需注意语义是否发生本地化演变。

##### 争议

一个值得关注的争议点在于：是否应依照地名所属国的发音进行音译？例如日本“関西”（Kansai），中文仍沿用汉字“关西”及其中文读音，而韩文已从관서（Gwan-seo）改为간사이（Kansai），以更贴近日语发音。韩国学界甚至呼吁中、日两国也互相改用对方语言的发音来称呼彼此地名。

然而，这一主张在表意文字体系的中文中推行难度较大。汉字地名的视觉识别性与历史延续性，使其难以被纯粹表音的形式取代。类似地，日、韩对“关羽”仍沿用各自传统读法（関羽 Kan'u / 관우 Gwan-u），并未改用中文发音。[①](https://www.bilibili.com/video/BV1GM411E7f8/)

所以尽管开头宣扬了世界主义，但是要完全消除这些差异却又是不符合客观规律的。在翻译实践中，译者宜遵循以下原则：

- **已有规范从规范**：对于已有通用译名（即使音译不完全准确）的地名、人名，应沿用既定译法，避免因追求“准确”而造成理解混乱。

- **尊重本国习惯**：如韩国提出的将泡菜 김치（kimchi）译作“辛奇”，虽有其考量，但因大众接受度有限，译者仍可沿用“泡菜”、“辣白菜”等既有译法。译名的更迭最终取决于公众语言习惯的自然演变，而非单方面推动。

- **在差异中寻求理解**：译者应在尊重历史习惯与促进文化交流之间保持平衡，既不必强行“纠正”既定译名，也应在必要时通过注释等方式增进读者对文化背景的理解。

### 走出去

部分译者在对外译介时，仍可能不自觉地受到一种隐性的文化位势差思维影响，习惯于以西方为中心的价值尺度来裁剪中国经验，甚至主动迎合某些“东方主义”的刻板想象。这种心态，在某种程度上可被视为历史上殖民文化霸权遗留的深层影响，尽管旧帝国已经被扫入了历史的垃圾桶中，但其文化霸权的影响仍潜藏于翻译的选材、措辞与视角之中。译者必须清醒认识到：**翻译不是单向的迎合，而是平等的对话**。在传递中国理念时，应避免将其简单归化为西方常见的政治口号，不能将“中华民族伟大复兴”草率对应为“Make America Great Again”，而应致力于呈现其独特的历史脉络、文化内涵与时代价值。

另一方面，翻译不是单向的文化灌输，更不是彰显“天朝上国”心态的渠道。若以居高临下之姿，忽视对象语言的文化语境、表达习惯与社会价值观，甚至轻视他者文化传统，则不仅违背我国一贯倡导的文明互鉴精神，更会引发受众反感，阻碍有效传播。翻译中的“文化傲慢”与“文化自卑”同样有害，皆不可取。

破而后立。真正的“走出去”应建立在**平等、尊重、深入理解**的基础之上，这就要求译者：

* 深度掌握译入语：译者不仅需要精通中文，更必须对目标语言及其文化语境有近乎母语者水平的理解与感知力。这包括其语言细微差别、文化典故、社会热点、当代思潮乃至潜在的敏感点。不了解目标受众，就无法实现有效传播。
* 秉持平等尊重之心：将译入语文化视为平等的对话伙伴，而非需要“教化”或“迎合”的对象。在传递中国价值观时，寻求既能准确达意、又符合目标语境接受习惯的创造性表达方式。
* 追求融合性创新：在忠实传达原文精神内核的前提下，灵活运用译入语的修辞资源、叙事方式与文化符号，进行“再语境化”的创造性转换，使中国故事既能保持本色，又能真正“入脑入心”。
